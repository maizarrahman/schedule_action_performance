## Schedule Action Performance

Add new fields in ir.cron named last_duration (duration of last execution in seconds) and last_status (success or failed).

Add new model named my.cron.log to store the execution log of ir.cron.

Add new views for my.cron.log to display the execution log in a tree and graph view.

Add new menu item named Schedule Action Log to display the execution log of ir.cron.
